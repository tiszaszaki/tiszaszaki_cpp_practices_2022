# Spiráltórusz genetátor példaprogram

## Használata

A generáláshoz szükséges paraméterek:
 1. "spiral_C_A"
 2. "spiral_C_B"
(spiral_C_B - spiral_C_A >= spiral_C_AB_diff_min)
 3. "spiral_C_r1" (> 0)
 4. "spiral_C_r2" (> 0)
 5. "spiral_C_N" (>= spiral_C_N_min)
 6. "spiral_C_menet_N"
 7. "spiral_kor_N" (>= spiral_kor_N_min)
